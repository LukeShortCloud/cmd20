# cmd20
### Version: 0.1.2

A free and open source dice roller CLI utility programmed in C.

## Usage
Install:
```
# gcc cmd20.c -o cmd20; mv cmd20 /usr/local/bin/
# mv cmd20.1 /usr/share/man/man1/
```

Example:
```
# cmd20.1 -n 82
Rolling a 82 sided die...
You rolled a 25
```

Help:
```
# cmd20 -h
OR
# man cmd20
```
